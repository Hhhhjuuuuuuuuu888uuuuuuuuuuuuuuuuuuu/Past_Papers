How to use your javascript to store information and retrive it in .txt file and .json file using javascript and htm
(solution1:Saving and Retrieving Text Files (.txt)
To create and download a .txt file, you can use JavaScript's Blob and URL.createObjectURL():

Save Data to .txt File
[
    HTML

    <textarea id="textInput" placeholder="Enter text here"></textarea>
    <button onclick="saveTextFile()">Save as .txt</button>

    JavaScript

    function saveTextFile() {
    let data = document.getElementById("textInput").value;
    let blob = new Blob([data], { type: "text/plain" });
    let anchor = document.createElement("a");

    anchor.href = URL.createObjectURL(blob);
    anchor.download = "data.txt";
    anchor.click();
}

This lets users type text into a field and save it as a .txt file
]

Saving and Retrieving JSON Files (.json)
[
Similarly, you can store data in JSON format and save it:
Save Data to .json File

JavaScript

function saveJsonFile() {
    let data = {
        name: "Colli",
        project: "Collins AI",
        interest: "AI development"
    };
    let blob = new Blob([JSON.stringify(data, null, 2)], { type: "application/json" });
    let anchor = document.createElement("a");

    anchor.href = URL.createObjectURL(blob);
    anchor.download = "data.json";
    anchor.click();
}

HTML

<button onclick="saveJsonFile()">Save as .json</button>
]

Reading Files (.txt and .json)
[
To read files, you can use the FileReader API

JavaScript

function readFile(event) {
    let file = event.target.files[0];
    let reader = new FileReader();

    reader.onload = function(e) {
        document.getElementById("output").textContent = e.target.result;
    };

    reader.readAsText(file);
}

HTML

<input type="file" onchange="readFile(event)">
<pre id="output"></pre>

This allows users to select a file and display its content
]

or saving them in a .json or .txt file. Keep in mind, this isn't secure—so don't use it in real applications!
(
{   Storing and Retrieving Credentials in localStorage
    
    Save Credentials
    [
    JavaScript

    function saveCredentials() {
    let username = document.getElementById("username").value;
    let password = document.getElementById("password").value; // Storing passwords like this is NOT secure

    let user = { username, password };
    localStorage.setItem("userData", JSON.stringify(user));
    alert("Credentials saved!");
    }
    ]

    Retrieve Credentials
    [
    JavaScript

    function retrieveCredentials() {
    let userData = JSON.parse(localStorage.getItem("userData"));
    if (userData) {
        alert(`Username: ${userData.username}\nPassword: ${userData.password}`);
    } else {
        alert("No credentials found!");
    }
   }
    ]

    HTML
    [
    <input type="text" id="username" placeholder="Enter username">
    <input type="password" id="password" placeholder="Enter password">
    <button onclick="saveCredentials()">Save</button>
    <button onclick="retrieveCredentials()">Retrieve</button>
    ]
}
