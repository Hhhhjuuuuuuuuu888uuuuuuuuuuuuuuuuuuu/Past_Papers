Introduction to Functions

A function is a fundamental concept in mathematics that describes a special relationship between inputs and outputs.

Definition: A function is a relation between a set of inputs (called the domain) and a set of permissible outputs (called the codomain) with the property that each input is related to exactly one output.

f: X → Y

We typically write y = f(x), where x is the input and y is the output.

Functions can be represented in multiple ways: algebraically (with formulas), graphically, numerically (with tables), or verbally (with descriptions).

Function Properties

Key Properties of Functions:

  • Domain: The set of all possible input values
  • Range: The set of all possible output values
  • One-to-one (Injective): Each output corresponds to only one input
  • Onto (Surjective): Every element in the codomain is mapped to by some element in the domain
  • Bijective: Both injective and surjective

Example: Consider the function f(x) = x²

  • Domain: All real numbers (-∞, ∞)
  • Range: All non-negative real numbers [0, ∞)
  • Not one-to-one (both 2 and -2 map to 4)
  • Not onto if codomain is all real numbers (negative numbers aren't outputs)

Types of Functions

Common Function Types:

  • Linear: f(x) = mx + b (straight line)
  • Quadratic: f(x) = ax² + bx + c (parabola)
  • Polynomial: Sum of terms with variables raised to powers
  • Rational: Ratio of two polynomials
  • Trigonometric: sin(x), cos(x), tan(x), etc.
  • Exponential: f(x) = aˣ
  • Logarithmic: f(x) = logₐ(x)

Function Composition: Combining two functions where the output of one becomes the input of another.

(f ∘ g)(x) = f(g(x))

Example: If f(x) = x² and g(x) = x + 1, then (f ∘ g)(x) = (x + 1)²

Practice Problems

Test your understanding of functions with these problems:

Problem 1: Determine whether each relation represents a function:

  1. {(1, 2), (2, 3), (3, 4), (1, 5)}
  2. {(x, y) | y = √x}

a) Not a function because the input 1 has two different outputs (2 and 5)

b) This is a function (for x ≥ 0) because each input has exactly one output

Problem 2: For f(x) = 2x + 3 and g(x) = x² - 1, find (f ∘ g)(2) and (g ∘ f)(2)

(f ∘ g)(2) = f(g(2)) = f(2² - 1) = f(3) = 2(3) + 3 = 9

(g ∘ f)(2) = g(f(2)) = g(2(2) + 3) = g(7) = 7² - 1 = 48

Problem 3: Find the inverse of the function f(x) = (2x + 5)/3

Steps to find the inverse:

1. Replace f(x) with y: y = (2x + 5)/3

2. Swap x and y: x = (2y + 5)/3

3. Solve for y: 3x = 2y + 5 → 2y = 3x - 5 → y = (3x - 5)/2

4. Write as f⁻¹(x): f⁻¹(x) = (3x - 5)/2
