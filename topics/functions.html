Introduction to Functions

A function is a fundamental concept in mathematics that describes a special relationship between inputs and outputs.

Definition: A function is a relation between a set of inputs (called the domain) and a set of permissible outputs (called the codomain) with the property that each input is related to exactly one output.

f: X ‚Üí Y

We typically write y = f(x), where x is the input and y is the output.

Functions can be represented in multiple ways: algebraically (with formulas), graphically, numerically (with tables), or verbally (with descriptions).

Function Properties

Key Properties of Functions:

  • Domain: The set of all possible input values
  • Range: The set of all possible output values
  • One-to-one (Injective): Each output corresponds to only one input
  • Onto (Surjective): Every element in the codomain is mapped to by some element in the domain
  • Bijective: Both injective and surjective
